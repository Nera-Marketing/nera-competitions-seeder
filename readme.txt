Seeds representative WooCommerce lottery competition products for development and demos.

== Description ==

**Nera Competitions Seeder** adds a Tools admin screen for seeding demo competition products that cover common Lottery for WooCommerce variants.

Seeded products include featured and gallery images when remote image sideloading is available, demo ticket data, and instant-win winner logs where relevant.

**Requires** WooCommerce. Lottery for WooCommerce should be active before seeding.

== Installation ==

1. Install and activate **WooCommerce**.
2. Install and activate **Lottery for WooCommerce**.
3. Upload and activate **Nera Competitions Seeder**.
4. Open **Tools > Nera Competitions Seeder** to seed or wipe demo competitions.
